## 🌊친구들의 응원과 함께하는 수영일기, Swimie!

🏊‍♂️ Swimie는 수영 기록을 간편하게 관리하고, 친구들과 응원을 주고받을 수 있는 커뮤니티 기반의 수영 기록 앱입니다. <br />
수영이 더 즐거워지는 공간, 스위미에서 오늘의 수영을 기록하고✒️, 친구들의 응원과 함께 동기부여를 얻어보세요! 💪

#### <a href="https://swimie.life">👉Swimie 바로가기</a>

<br />
